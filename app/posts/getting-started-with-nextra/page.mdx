# Getting Started with Nextra

Nextra is a powerful static site generator built on top of Next.js. It's perfect for creating documentation sites and blogs.

## Why Choose Nextra?

- **Built on Next.js** - Get all the benefits of React and Next.js
- **MDX Support** - Write content in Markdown with React components
- **Theme System** - Beautiful, customizable themes out of the box
- **Performance** - Static generation for lightning-fast sites
- **SEO Friendly** - Built-in SEO optimizations

## Features

### Code Syntax Highlighting

interface BlogPost {

### Built-in Components

Nextra comes with many useful components like callouts, tabs, and more.
