## Overview

Ad Genie is a simple web application that leverages Google's cutting-edge Gemini 2.5 Flash Image model to automatically generate professional advertisements from your product photos. Simply upload an image, choose a theme, and watch AI create a stunning marketing visual tailored to your needs.

## Key Features

### AI-Powered Ad Generation
- Utilizes **Google Gemini 2.5 Flash Image** for high-quality advertisement creation
- Transforms product images into themed advertisements using AI
- Supports various image formats (PNG, JPG, WEBP) up to 10MB

### Multiple Theme Options
Choose from 9 professionally designed themes:
- **Minimalist** - Clean, elegant designs with plenty of white space
- **Festive** - Holiday and celebration-themed advertisements
- **Lifestyle** - Real-life scenarios showcasing product usage
- **Luxury** - Premium, high-end aesthetic for luxury products
- **Office** - Professional business environment styling
- **Nature** - Organic, eco-friendly themes with natural elements
- **Summer** - Bright, vibrant summer-inspired designs
- **Bold** - High-impact visuals with striking colors
- **Sale** - Promotional designs with urgency and value messaging

### Intuitive User Experience
- Drag-and-drop image upload interface
- Real-time preview of uploaded images
- Theme selection with visual previews
- One-click download of generated advertisements
- Recent ads history for easy access

### Technical Excellence
- Built with **React 19** and **TypeScript** for type-safe development
- **Vite** for lightning-fast development and building
- **Tailwind CSS** for responsive, modern styling
- **React Router** for seamless navigation
- **React Dropzone** for enhanced file upload experience

## Google Gemini 2.5 Flash Image Integration

Ad Genie harnesses the power of **Google's Gemini 2.5 Flash Image** model, also known internally as **Google Nano Banana** during development. This state-of-the-art multimodal AI model excels at understanding visual content and generating high-quality images based on textual descriptions and visual inputs.

### Why Gemini 2.5 Flash Image?

- **Multimodal Capabilities**: Processes both text and images simultaneously
- **High-Quality Output**: Generates professional-grade advertisement images
- **Fast Generation**: Optimized for quick response times
- **Contextual Understanding**: Analyzes product images to create relevant ad content
- **Theme Adaptation**: Applies different styling themes based on user selection

The integration uses Google's Generative Language API to:
1. Use the uploaded product image as a reference
2. Generate themed prompts based on user selection
3. Create professional advertisements that highlight the product effectively

## Getting Started

### Prerequisites
- Node.js 18+ installed
- Google AI Studio API key ([Get one here](https://aistudio.google.com/app/apikey))

### Installation

```bash
# Clone the repository
git clone https://github.com/imSuma/ad-genie.git
cd ad-genie

# Install dependencies
npm install

# Start development server
npm run dev
```

### Configuration

1. **Get Google AI Studio API Key**
   - Visit [Google AI Studio](https://aistudio.google.com/app/apikey)
   - Create a new API key
   - Copy the key for use in the application

2. **Configure API Key in App**
   - Open the application
   - Click on the Settings icon
   - Paste your API key in the configuration modal
   - The key is stored securely in your browser's local storage

### Usage

1. **Upload Product Image**
   - Drag and drop your product photo or click to browse
   - Supported formats: PNG, JPG, WEBP (max 10MB)
   - Best results with high-resolution images and clear product visibility

2. **Select Advertisement Theme**
   - Choose from 9 available themes
   - Each theme is optimized for different marketing contexts
   - Preview theme descriptions before selection

3. **Generate Advertisement**
   - Click "Generate Ad" to start the AI process
   - Wait for Gemini 2.5 Flash Image to create your advertisement
   - Download the generated image for immediate use

## Best Practices for Optimal Results

### Image Guidelines
- **High Resolution**: Use images with at least 800px width for best results
- **Good Lighting**: Ensure your product is well-lit and clearly visible
- **Simple Backgrounds**: Plain or minimal backgrounds work best for AI processing
- **Clear Focus**: Make sure your product is the main focus of the image
- **Square/Portrait**: These orientations typically work best for social media ads

### Theme Selection Tips
- **Minimalist**: Perfect for tech products, jewelry, or clean lifestyle brands
- **Luxury**: Ideal for high-end products, fashion, or premium services
- **Lifestyle**: Great for everyday products shown in real-world contexts
- **Bold**: Excellent for sports equipment, energy drinks, or dynamic products
- **Sale**: Best for promotional campaigns and limited-time offers
